# Demo 演示路径

1. 打开 `/mobile`，展示橘橘当前状态。
2. 打开 Electron 桌宠，确认桌面角落同步显示。
3. 打开 `/demo`，切换到“喝水”。
4. 手机 H5 和桌宠同步变成喝水。
5. 在聊天里问：“你现在在干嘛？”
6. AI 回答引用“刚喝完水”。
7. 切换到“跑步”或“异常活跃”，展示动画变化。
8. 打开 `/memories`，上传一张图片或填写一条文字回忆。
9. 页面显示“AI 分析中”，随后写入时间线。
10. 问：“你还记得生日那次吗？”或“你还记得第一次见面吗？”
11. AI 引用回忆回答。
12. 打开 `/eval` 展示评测报告卡片。

兜底：

```text
硬件不稳 -> 使用 /demo 手动切状态
智谱网络不稳 -> 切 mock 或 cached
手机无法局域网访问 -> 使用 tunnel
上传素材异常 -> 用“只新增文字”
```
