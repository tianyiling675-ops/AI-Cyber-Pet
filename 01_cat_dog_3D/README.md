# pet_necklace

宠物智能健康项圈 3D 交互演示。

## 项目内容

- `pet-collar-3d.html`：主交互页面，包含一体式硬项圈 / 独立夹扣模块两种形态。
- `modules/detail.html`：单个传感器模块的 3D 特写、爆炸图、技术参数和设计原理。
- `modules/internals.html`：一体式净面圆环的内部结构爆炸页。
- `modules/data.js`：共享外观风格、传感器参数、SVG 图纸和尺寸逻辑。
- `3D_eg_/`：产品调研与双形态设计文档。
- `3D_eg_/项圈模型尺寸与工程参数.md`：一体式项圈的半径、宽度、厚度、开合和工程参数。
- `ref_internal.png` / `ref_small.jpg`：内部结构参考图。
- `vendor/three/`：本地 Three.js 运行依赖，避免 CDN 网络不可用时页面空白。

## 本地运行

由于页面使用 ES modules，请通过本地 HTTP 服务打开，不建议直接双击 HTML：

```powershell
python -m http.server 5173 --bind 127.0.0.1
```

然后访问：

```text
http://127.0.0.1:5173/
```

## 已实现

- 6 套外观风格切换。
- 16-60cm 颈围尺寸调节，默认 56cm / 60% 带宽，并提供 52/54/56/58/60cm 快捷尺度方案。
- 一体式硬项圈：外观净面，底部 C 型开合可拖动，点击圆环进入内部结构页。
- 一体式硬项圈：内置柔性伸缩叠带，外观不暴露模块或滑块。
- 夹扣式模块：单体胶囊夹在普通项圈上，支持安装详图/爆炸视图。
- 5 个功能模块：主控舱、心率、体温、定位、运动姿态。
- 模块详情页与主页面状态同步。
- 截图导出与 GLB 模型导出。
