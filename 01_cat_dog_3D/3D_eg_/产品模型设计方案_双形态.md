# 宠物智能项圈 · 产品模型设计方案（双形态）

> 文档类型：产品设计方案 / Product Design Spec
> 日期：2026-06-14　｜　关联项目：`pet-collar-3d.html` 3D 演示
> 适用范围：**两款产品形态的完整设计** —— 形态 A 一体式硬项圈、形态 B 独立夹扣模块

---

## 一、产品定位与双形态策略

同一套传感器技术与算法内核，做成**两种形态**覆盖两类需求：

| | **形态 A：一体式硬项圈** | **形态 B：独立夹扣模块** |
| --- | --- | --- |
| **一句话定位** | 把所有功能焊死在项圈里，只调大小 | 一个小盒子，夹到任何项圈上 |
| **核心卖点** | 全功能、医疗级精度、佩戴一体感 | 通用、可移植、不挑项圈 |
| **目标用户** | 重度养宠、愿为健康监测付费、要颜值 | 已有好项圈、想要 GPS+健康、预算敏感 |
| **价格带** | 中高端（对标 PetPace / Fi） | 入门~中端（对标 Tractive / Invoxia） |
| **对标产品** | PetPace 3.0、Fi Series 3+ | Tractive DOG 6、Invoxia、AirTag |

**关键洞察**：两种形态不是简单"贵版/便宜版"，而是**传感器技术路线不同**——
- 形态 A 内嵌贴颈，用**接触式**传感器（PPG + 红外），精度可以做得很高；
- 形态 B 外挂在项圈上方，**碰不到皮肤**，所以心率/呼吸改用**雷达式**（穿透毛发+项圈带），体温改用**双通道估算**。

这是 Invoxia 能用 €100 干翻 PetPace €300 的核心逻辑，也是本方案两条产品线的差异化根基。

---

## 二、共享技术底座（5 个功能模块）

两款形态共用同一套**功能定义**（与现有 `data.js` 的 5 个模块一一对应），只是**实现位置和传感器原理**不同：

| 模块 | 功能 | 形态 A（接触式） | 形态 B（非接触式） |
| --- | --- | --- | --- |
| **pod 主控** | MCU + 电池 + 通信 | nRF5340 + 300mAh + BLE 5.3 | nRF5340 + 220mAh + BLE 5.3 / LTE-M |
| **heart 心率** | 心率/呼吸/HRV | **PPG 光电**（绿光525+红外880nm，贴肤） | **毫米波雷达**（穿透毛发+项圈带，不贴肤） |
| **temp 体温** | 体表温度 | **红外热电堆**（贴肤，±0.1℃） | **双通道估算**（雷达体温趋势 + 环境补偿，±0.3℃） |
| **gps 定位** | 位置/围栏/轨迹 | GPS+北斗+GLONASS L1/L5 | GPS+北斗+GLONASS + LTE-M 蜂窝回传 |
| **motion 运动** | 步态/睡眠/姿态 | 6 轴 IMU（BMI270） | 6 轴 IMU（BMI270） |

> **一致性原则**：对外（App、品牌、数据格式）两款产品体验完全统一；对内（硬件）按形态选传感器。这样一套 App、一套算法、一个品牌支撑两条产品线。

## 三、形态 A：一体式硬项圈（详细设计）

### 3.1 设计理念

"**功能焊死在项圈里，外观只剩项圈本身。**" —— 用户看到的不是"项圈 + 一堆疙瘩"，而是一条干净的、有设计感的项圈。所有电子件藏在带体内部，只露出必要的传感窗口和充电/按键。

对应昨天 `pet-collar-3d.html` 已做的 6 种风格（竞速机能 / 极简纤薄 / 萌宠泡泡 / 复古铆钉 / 六角装甲 / 花瓣环绕），本形态是它的产品化落地。

### 3.2 结构设计（模块如何藏在内部）

项圈横截面是**多层复合结构**，从外到内：

```
            ┌──── 外饰层（硅胶包胶 / 皮革 / 编织尼龙，可换风格）
            ├──── 防护层（凯夫拉纤维编织网，防咬断）
            ├──── FPC 柔性排线层（天线 + 传感器走线，随带体弯曲）
            ├──── 刚性主控仓（PC+ABS，IP67 密封，内含 MCU/电池/GPS/IMU）
            └──── 贴肤传感层（PPG 窗口 + 红外温区 + 梳齿状毛发导槽）
                   ↑ 朝向皮肤一侧
```

**关键结构点**：
- **主控仓居中**：最厚的部分放在项圈正后方（对应宠物后颈），这里活动量小、不易磕碰，且重心居中防偏转。
- **传感窗口靠前**：PPG + 红外布置在项圈**内侧贴合颈动脉**的位置，窗口做**梳齿状/凹陷式**把毛导开（长毛宠物关键）。
- **GPS 天线朝上**：贴在带体外侧顶部，远离地面，信号最好。
- **无外挂疙瘩**：所有东西都在带体厚度内（目标带厚 ≤ 8mm），外观是一条连贯的带子。

### 3.3 尺寸调节机制（本形态的核心要求："只调大小"）

既然模块焊死、不能拆，**调大小**就是唯一可动结构。给出三套方案，按优先级：

#### 方案 ①：无级棘轮调节（首选 ⭐）
- **原理**：类似运动手表的**锯齿滑轨**——带体一端是齿条，另一端是带弹簧卡爪的棘轮扣。
- **操作**：拉紧即自动锁止（"咔咔"声），按下释放键松开。
- **优点**：无级调节（覆盖整个颈围区间）、**没有多余的带尾外露**（外观最干净）、单手可操作、戴好不会自己松。
- **区间**：单尺寸覆盖 S（16–26cm）/ M（24–38cm）/ L（36–55cm）三档产品，每档内部无级。
- **风险**：模具成本高，棘轮件要过寿命测试（≥10000 次锁止）。

#### 方案 ②：快拆插销 + 多定位孔（备选）
- **原理**：传统皮带孔思路的升级版——不锈钢插销插入不同孔位。
- **优点**：结构简单、可靠、成本低。
- **缺点**：调节有级（孔间距 1cm）、多余带尾需折叠固定、不如棘轮优雅。

#### 方案 ③：弹性伸缩带（辅助）
- 用于幼猫/小型犬，靠 TPE 弹性带自适应贴合，不靠机械锁止。
- 适合"套头即戴"的极简体验，但精度依赖弹性一致性。

**推荐组合**：M/L 款用**棘轮**，S 款（猫）用**弹性 + 棘轮双保险**。

### 3.4 材质与工艺

| 部件 | 材料 | 工艺 |
| --- | --- | --- |
| 外饰带体 | 亲肤硅胶 / 编织尼龙 / 真皮（按风格切换） | 二次注塑 / 缝合 |
| 防护层 | 凯夫拉纤维编织网 | 层压复合 |
| 主控仓 | PC+ABS | 超声波焊接密封 |
| 传感触点/窗口 | 316L 不锈钢 / 蓝宝石玻璃 | CNC + 盐雾测试 48h |
| 调节扣 | 钛合金 / 锌合金 | 压铸 + PVD 镀膜 |
| 充电 | 磁吸 pogo pin | 无开口，IP68 |

**重量红线**：整条 ≤ 35g（猫/小型犬款）、≤ 60g（中大型犬款）。

### 3.5 佩戴与准确性

- **佩戴位置**：耳下高位，传感窗贴颈动脉侧。
- **松紧**：两指法则；棘轮方案可精确锁到"两指"压力，**接触压力 5–15 g/mm²**（这是 PPG/红外测准的硬指标）。
- **精度**：接触式贴肤 → 心率 ±2 BPM、体温 ±0.1℃、GPS ≤5m，**可达兽医级**（对标 PetPace）。

### 3.6 优缺点

| ✅ 优点 | ❌ 缺点 |
| --- | --- |
| 外观最干净，真正"配饰感" | 模块焊死，损坏需整条换 |
| 接触式精度高，可做医疗级 | 模具/研发成本高 |
| 佩戴一体感强，宠物无异物感 | 用户需选对尺寸（S/M/L） |
| 防水防咬做得最好（全密封） | 无法换风格（除非换整条） |

## 四、形态 B：独立夹扣模块（详细设计）

### 4.1 设计理念

"**一个小盒子，夹到任何项圈上。**" —— 用户已有的尼龙项圈、皮质项圈、甚至是胸背带，都能直接夹上用。模块本身是**独立完整的产品**，项圈只是它的"载体"。

对标 Tractive / Invoxia / AirTag 的形态，但用一个模块同时做 GPS + 健康（这是 Tractive 做不到、Invoxia 做到了的方向）。

### 4.2 结构设计（单一模块）

模块是一个**硬币/胶囊形小盒**，分上下两部分：

```
        ┌─────────────────────┐  ← 上盖（朝外）：GPS 天线、LED、Logo
        │   主控 + 电池 + GPS  │
        │   + IMU + 雷达天线  │
        ├─────────────────────┤  ← 中缝：自适应夹口（夹住项圈带）
        │   弹簧夹 + 硅胶垫   │     ← 夹持区，开口 15–25mm 可调
        └─────────────────────┘  ← 下底（朝皮肤侧）：雷达辐射面
                                   （不贴肤，靠雷达穿透项圈带 + 毛发）
```

**关键结构点**：
- **中缝夹口**：模块从中间"咬住"项圈带，像夹子/书夹。夹口内衬**硅胶防滑齿**，不伤项圈、不滑脱。
- **夹口宽度自适应**：弹簧机构兼容 **15–25mm 宽**的带体（覆盖市面上绝大多数尼龙/皮质项圈）。
- **下底朝皮肤**：雷达辐射面朝下，信号穿透夹住的项圈带 + 毛发，到达宠物颈部皮下血管，测心率/呼吸。
- **无触点**：不依赖贴肤，所以**不怕长毛、不怕戴歪**，鲁棒性远高于形态 A。

### 4.3 夹持适配机制（本形态的核心要求："装在各种项圈上"）

给出三种适配方案，按优先级：

#### 方案 ①：自适应弹簧夹（首选 ⭐）
- **原理**：类似强力长尾夹/书夹，**按压尾部张开夹口，松开自动夹紧**。
- **夹口**：硅胶防滑齿 + 弹簧，适配 15–25mm 宽、2–6mm 厚的任意带体。
- **兼容**：尼龙编织带 ✅、真皮带 ✅、织带胸背 ✅、链条项圈 ❌（链条夹不住，需另配挂环）。
- **优点**：秒装秒拆、不挑项圈、牢固（脱拔力 ≥ 50N）。
- **风险**：弹簧疲劳 → 需选高循环寿命弹簧钢，过 10000 次夹持测试。

#### 方案 ②：硅胶滑套（备选，Tractive 同款）
- **原理**：模块底部连一个**硅胶套**，把项圈带穿过去。
- **优点**：结构最简单、成本最低、防水好。
- **缺点**：需要把项圈从宠物头上取下才能穿（不如弹簧夹方便）、只适配固定宽度。

#### 方案 ③：尼龙绑带 + 魔术贴（辅助）
- 一根细尼龙带绕过项圈 + 模块，魔术贴固定。
- 兼容性最广（连链条都行），但最不优雅。

**推荐组合**：**默认弹簧夹**（方案①），附赠一个硅胶滑套（方案②）作为链条/异形项圈的适配。

### 4.4 材质与工艺

| 部件 | 材料 | 工艺 |
| --- | --- | --- |
| 模块外壳 | PC+ABS / 铝合金（高端款） | 注塑 / CNC 阳极氧化 |
| 夹口硅胶垫 | 食品级硅胶（Shore 50A） | 模压成型 |
| 弹簧 | 304 不锡钢弹簧丝 | 冷卷 + 回火 |
| 雷达窗口 | 玻纤增强 PC（雷达透过率好） | 注塑 |
| 充电 | 磁吸 pogo pin | IP68 |

**重量红线**：模块本体 ≤ 20g（对标 Tractive 30g、Invoxia 更轻），轻到宠物无感。

### 4.5 佩戴与准确性

- **佩戴**：夹在项圈**正下方**（对应宠物喉部/下颈），这里雷达信号到颈部大血管最近。
- **松紧**：**不依赖松紧**——这是形态 B 相对形态 A 的最大优势。只要夹住不掉，雷达就能测。
- **精度**：雷达非接触 → 心率/呼吸靠 AI 算法（对标 Invoxia，论文实测 **HR 99.6% / RR 98.6%**）；GPS ≤5m。**体温为趋势级**（±0.3℃，不如形态 A 的 ±0.1℃），用于发烧预警而非诊断。

### 4.6 优缺点

| ✅ 优点 | ❌ 缺点 |
| --- | --- |
| 不挑项圈，秒装秒拆 | 非接触，体温精度不如形态 A |
| 模块独立，损坏只换模块 | 多一个"疙瘩"在项圈上 |
| 不怕长毛、不怕戴歪 | 链条项圈需额外适配 |
| 成本低、研发周期短 | 夹持可靠性依赖弹簧寿命 |
| 用户门槛低（已有项圈即可用） | 外观不如形态 A 一体 |

## 六、传感器路线差异（为什么 A 接触、B 雷达）

这是整套设计的**技术核心**，值得单独讲清：

### 6.1 形态 A 为什么用接触式（PPG + 红外）
- 形态 A 内嵌贴颈，**传感器天然贴在皮肤上**，不用白不用。
- 接触式 PPG/红外精度上限高（体温 ±0.1℃），能撑起"医疗级"定位和溢价。
- 缺点是**怕松、怕毛**——所以配套棘轮调紧 + 梳齿导毛。

### 6.2 形态 B 为什么必须用雷达
- 形态 B 夹在项圈**上方**，和皮肤之间隔着**项圈带 + 毛发**，接触式根本测不到。
- 雷达（毫米波）能**穿透这些介质**直达皮下血管微动，这是唯一能让"外挂模块"也测心率的方案。
- 代价：体温只能做趋势级（雷达对温度不敏感，靠算法估算 + 环境补偿）。
- 收益：**不怕长毛、不怕戴歪、不挑项圈**——鲁棒性碾压接触式。

> **这正好打中两种用户**：要精度的选 A（接触式），要省心通用的选 B（雷达式）。两条线技术互补、不打架。

## 七、3D 演示落地建议

现有 `pet-collar-3d.html` 已实现了**形态 A**（6 风格一体式）。建议在演示中**增加一个形态切换**，让两种形态都能展示：

### 7.1 数据层（data.js）扩展
```js
// 新增：形态定义
export const FORMS = {
  integrated: { name:'一体式硬项圈', desc:'功能焊死在带体内，棘轮调节大小' },
  clip:       { name:'独立夹扣模块', desc:'单一模块，弹簧夹适配任意项圈' },
};
```

### 7.2 3D 场景
- **形态 A 视图**（已有）：环形带体 + 沿环分布的 5 个传感标注 + 棘轮扣。
- **形态 B 视图**（新增）：一条普通尼龙项圈（简化环）+ 夹在底部的胶囊模块 + 模块爆炸图（拆出雷达天线/GPS/电池/弹簧夹）。
- **切换交互**：主页加一个「一体式 / 夹扣式」切换按钮，切换时带体 morph 或淡入淡出。

### 7.3 详情页 principle 文案（直接可用）

**形态 A（接触式）principle 示例**：
> 内嵌贴颈设计，PPG 光电传感器紧贴颈动脉皮肤，配合医疗级红外热电堆，实现 ±0.1℃ 体温与 ±2 BPM 心率的诊断级精度。棘轮无级调节锁定"两指"松紧，保证接触压力恒定在 5–15 g/mm² 的黄金区间。

**形态 B（雷达式）principle 示例**：
> 独立胶囊模块通过自适应弹簧夹固定在任意项圈下方。内置毫米波雷达穿透项圈带与毛发，非接触捕捉颈部血管微动，经 AI 算法还原心率/呼吸（实测对标金标准 HR 99.6% / RR 98.6%）。不挑项圈、不怕长毛、不依赖佩戴松紧。
